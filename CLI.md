# Command Line Interface (CLI)

## Create Project

```commandline
pfunk init <project_name>
```
## Add Stage

```commandline
pfunk add_stage <stage_name>
```

## Publish

```commandline
pfunk publish <stage_name>
```
## Create Admin User

```commandline
pfunk create_admin_user <stage_name>
```

## Seed Keys

```commandline
pfunk seed_keys <stage_name>
```

## Deploy

```commandline
pfunk deploy <stage_name>
```

## Run Server

```commandline
pfunk local 
```
